## ðŸŽ¬ Video Structure (90 seconds)

### Scene 1: Hook (0-10s)
**Visual**: Dashboard with some data
**Script**: "Every AI prompt has a hidden cost. Energy. Carbon. Water. EcoTrace makes it visible."

**Actions**:
- Show dashboard with existing data
- Pan across the stat cards
- Highlight the numbers

### Scene 2: The Problem (10-20s)
**Visual**: ChatGPT interface
**Script**: "When you ask ChatGPT a question, you never see the environmental impact."

**Actions**:
- Show ChatGPT interface
- Type a prompt slowly
- Pause before submitting

### Scene 3: The Solution (20-45s)
**Visual**: Prompt submission + notification
**Script**: "EcoTrace changes that. Watch what happens when I submit this prompt."

**Actions**:
1. Submit prompt: "Explain quantum computing"
2. Wait for response to complete
3. Notification appears (highlight this!)
4. Point to each metric:
   - "0.2 Watt-hours of energy"
   - "0.1 grams of CO2"
   - "0.1 milliliters of water"
5. Show the green tip

### Scene 4: Dashboard (45-60s)
**Visual**: Switch to dashboard tab
**Script**: "Every prompt is tracked in real-time on your dashboard."

**Actions**:
1. Switch to dashboard tab
2. Show updated totals
3. Point to the chart
4. Highlight the new data point
5. Show prompt counter increased

### Scene 5: Comparison (60-75s)
**Visual**: Submit longer prompt
**Script**: "Let's compare. Here's a much longer prompt."

**Actions**:
1. Switch back to ChatGPT
2. Submit: "Write a detailed 500-word essay about climate change"
3. Show notification with higher numbers
4. Point out the difference: "3x more energy!"
5. Show green tip: "Try shorter prompts"

### Scene 6: Call to Action (75-90s)
**Visual**: Dashboard with all data
**Script**: "Make your AI usage visible. Make it sustainable. Try EcoTrace."

**Actions**:
1. Show final dashboard state
2. Pan across all features
3. Show extension icon
4. End with logo/title card

## ðŸŽ¯ Recording Tips

4. Ensure smooth transitions
